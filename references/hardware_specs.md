# Orange Pi Zero 2W Hardware Specifications

## Overview
The Orange Pi Zero 2W is a compact, high-performance single-board computer (SBC) designed for embedded applications and IoT projects.

## Core Components
- **SoC**: Allwinner H618
  - **CPU**: Quad-core Cortex-A53 processor @ 1.5GHz
  - **GPU**: Mali G31 MP2 (Supports OpenGL ES 1.0/2.0/3.2, OpenCL 2.0, Vulkan 1.1)
- **RAM**: 4GB LPDDR4
- **Storage**: 
  - MicroSD card slot (Current setup: 32GB Lexar SD card)
  - SPI Flash: 16MB

## Connectivity
- **Wi-Fi/Bluetooth**: Dual-band Wi-Fi (2.4GHz and 5GHz) and Bluetooth 5.0 (CDTech 20U5622)
- **Video Output**: Micro HDMI 2.0 up to 4K@60fps
- **USB**: 
  - 1x USB Type-C (Power supply + USB 2.0 OTG)
  - 2x USB Type-C (USB 2.0 via expansion board or direct)
- **GPIO**: 24-pin header (compatible with Raspberry Pi Zero 2W form factor)

## Power
- **Input**: 5V/2A or 5V/3A via USB Type-C

## Operating System
- **Current OS**: Ubuntu 24.04 LTS (Noble Numbat)
